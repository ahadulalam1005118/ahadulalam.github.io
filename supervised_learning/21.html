Algorithm for decision tree
learning

Basic algorithm (a greedy
divide-and-conquer
algorithm)

Assume attributes are categorical now (continuous attributes
can be handled too)

Tree is constructed in a
top-down recursive manner

At start, all the training examples are at the root

Examples are partitioned recursively based on selected
attributes

Attributes are selected on the basis of an impurity function
(e.g.,
information gain
)

Conditions for stopping partitioning

All examples for a given node belong to the same class

There are no remaining attributes for further partitioning –
majority class is the leaf

There are no examples left
